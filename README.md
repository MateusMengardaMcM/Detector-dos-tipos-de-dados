
# Detector de tipos de dados

Projeto voltado para fins educacionais, com intenção de aprender e compreender os tipos de dados e como identificá-los.


## Aprendizados

Compreendi quais são os tipos de dados e como identificá-los.
## Funcionalidades

- Ao inserir um texto e clicar no botão, o computador identificará e apresentará o tipo de dado, entre string, number e boolean.

## Stack utilizada

**Front-end:** Html, css & JavaScript.



## Deploy

Para fazer o deploy desse projeto faça um clone do repositório e execute o arquivo em index.html.

```bash
  git clone https://github.com/MateusMengardaMcM/Detector-dos-tipos-de-dados
```
